## Layout Hierarchy

### Full PDF Structure (Group Entry):

1. **HEADER SECTION** (Gradient teal/cyan)
   - TOPAZ 2.0 title
   - "Dance Competition" subtitle
   - "Official Score Sheet"
   - "Heritage Since 1972"

2. **COMPETITION INFO CARD** (Light gray box)
   - Competition name
   - Date and venue
   - Judge count and entry number

3. **COMPETITOR INFO SECTION** (Colored by rank)
   - Rank badge (if applicable)
   - Entry photo (if available)
   - Group name and age
   - **Studio & Teacher** (if present) â­ NEW
   - Category badges (category, age division, ability)
   - Category rank (if applicable)
   - Average score (if available)

4. **GROUP MEMBERS SECTION** (Light gray box) â­ NEW
   - "GROUP MEMBERS:" header
   - Bulleted list of all members
   - Ages shown when available

5. **DETAILED SCORES TABLE**
   - Individual judge scores
   - Technique, Creativity, Presentation, Appearance
   - Total scores per judge
   - Average row (bold, colored)

6. **CATEGORY ANALYSIS TABLE**
   - Comparison across judges
   - Category averages

7. **JUDGE NOTES** (Yellow boxes)
   - Individual notes per judge
   - Only if notes were entered

8. **MEDAL PROGRAM STATUS** (Yellow box)
   - Only if enrolled in medal program
   - Current medal level and points

9. **FOOTER**
   - Copyright line
   - Heritage tagline
   - Official results text
